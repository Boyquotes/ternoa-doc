 ## Please read steps before start

#### 1. Clone depot git on your local env : 
```
git clone https://github.com/capsule-corp-ternoa/ternoa-doc.git
```

#### 2. Please use yarn start to open in your Localhost : 
```
yarn start
```

#### 3. change your file `.md` in `.mdx` file.
**For exemple** : I want to add infos in `for-developers/cookbooks/create-a-capsule.md`
I should to change file in `for-developers/cookbooks/create-a-capsule.mdx`

#### 4. Copy/Past common template
- Copy all are in  `start-your-doc.mdx`
- ðŸ”¥ **Be careful** don't touch metadatas in top of all docs. It's already done. 
```jsx title="for-developers/cookbooks/create-a-capsule.mdx"

---
```
- Past directly in your `docname.mdx`
- Bring modifications in your `docname.mdx`

#### 5. Create a new branch from `update/mdxDocs@NET-` 
**example** : update/create-a-capsule@

### Here you can add a part of code with path
Use ` ``` your code ``` ` to generate sample of code with color synthax.

```jsx title="ternoa-js/account"
// Example
import { generateSeed, getKeyringFromSeed } from "ternoa-js/account";

(async () => {
  const account = await generateSeed()
  const keyring = await getKeyringFromSeed(account.seed)
  const address = keyring.address
  console.log("Your fresh public address is: ", address)
})().catch((e) => {
  console.log(e)
})
```
